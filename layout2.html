Media Query

Media queries is a feature of CSS 3 allowing content rendering to adapt to different conditions such as screen resolution. It became a W3C recommended standard in June 2012, and is a cornerstone technology of responsive web design.

@media only all and (condition) {implementation}

  • @media: is a starting point of media query.
  • only: prevents older browsers that do not support media queries with media features from applying the specified styles.
  • all: is used by default when no other type is specified.
    • all: used for all media type devices.
    • print: used for printers.
    • screen: used for computer screens, tablets, smart-phones etc.
    • aural: used for screen readers.
    • tv: used for television.
    • handheld: used for handheld devices.
    • projection: used for projector.
  • and: The and keyword combines a media feature with a media type or other media features.
  • (condition): boolean condition
  • {implementation}: provides implementation based on conditions.
